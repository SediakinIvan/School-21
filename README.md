# School 21 - Проекты по ИИ и машинному обучению

Этот репозиторий содержит коллекцию проектов, выполненных в рамках обучения в School 21, посвященных работе с искусственным интеллектом, языковыми моделями и созданию интеллектуальных агентов.


### [Project1 - Классификация учебных материалов](./Project1/)
Система классификации учебных материалов по предметам с использованием GigaChat API. Включает три варианта реализации:
- Базовая классификация с простым промптом
- Структурированный вывод с Pydantic
- Интеграция с инструментами для сохранения данных

### [Project2 - Telegram бот для классификации](./Project2/)
Telegram-бот для классификации учебных материалов с расширенным функционалом:
- Классификация ссылок по предметам
- Генерация отчетов по периодам
- Сохранение и фильтрация данных
- Интеграция с LangGraph для управления состоянием

### [Project3 - Агент для создания резюме](./Project3/)
Интеллектуальный агент для создания резюме и мотивационных писем:
- Пошаговый сбор информации о кандидате
- Адаптация под конкретную стажировку
- Поддержка различных стилей оформления
- Возможность редактирования документов
- Telegram-интерфейс


- **Python 3.13+**
- **GigaChat API** - российская языковая модель
- **LangChain** - фреймворк для работы с LLM
- **LangGraph** - создание агентов с управлением состоянием
- **Pydantic** - валидация данных
- **Telegram Bot API** - интерфейс для ботов
- **JSON** - хранение данных


1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd "School 21"
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Настройте API ключи в соответствующих файлах проектов

5. Запустите нужный проект:
```bash
# Project1
cd Project1
python project01_task01.py

# Project2
cd Project2
python project02_task02.py

# Project3
cd Project3
python resume_agent_improved.py
```


- Python 3.13+
- API ключ GigaChat
- Telegram Bot Token (для Project2 и Project3)
- Интернет-соединение
